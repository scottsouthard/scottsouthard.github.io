CSS Positioning

One of the more difficult CSS concepts to fully wrap your head around is positioning. Since we're not actively picking up pieces of text or images and moving them around, there are rules in the code that determine how each object acts on its own, in relation to the other objects, and in relation to the page.

The four main position properties are:

  • Static
  • Relative
  • Absolute
  • Fixed
Static positioning is the most basic property, as an element that is static simply follows the general flow of the HTML document. It shows up where it naturally would. Static is the default setting for HTML elements, and cannot be altered by left, right, top, and bottom properties.



Relative positiong is slightly more complicated, but really, it's just the upgraded version of a static element. An element with a relative position can be moved, and the movements that are made are always relative to the standard position.



Absolute positioning takes everything one step further down the rabbit hole and bases the position of the element off of it's closest parent. So rather than plopping it down naturally (like static) or relative to its natural place (like relative), absolute positioning is relative to another element. This way, you can group elements together, placement-wise, and always know that they'll be connected to each other.



Fixed positioning is it's own beast. While the rest of these properties are based on the HTML document or the elements within, fixed positioning is based solely on the viewport of the user (that means your screen). So when you scroll through a page or move left or right, a fixed element will remain planted firmly in place, not moving with the rest of the elements on the page.


The concepts of positioning seem obscure and obtuse at first, but like everything under the hood of the internet, there's a straightforward, easily understandable answer behind it all, you just have to take a moment to think it through. With all four of these concepts in the CSS repertoire, the possibilities for positioning are endless.
