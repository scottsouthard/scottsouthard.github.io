Cheat Sheet

Ruby Enumerables and Ephemera

One of the most important facets of learning any language is having a good grip on the terms, the vocabulary if you will, of language and being able to implement it with fluidity and efficiency. But there are A LOT of methods, objects, etc in Ruby, and memorizing all of them is damn near impossible. I've put together a small cheat sheet for the Enumerable module (iterators that go through collections using each), just so we have a go-to for some of the most widely usable iterators.

#any?
Passes the collection through a block of code and returns a boolean true or false. If one or more return true, the enumerator returns true.

#count
Returns the number of items in the collection. Can be passed a block of code with specific parameters.

#detect
Similar to #any? except #detect returns the first entry that is not false, rather than a true or false.

#each_with_index
Passes each item and it's index through a block. Useful for figuring out where entries are and swapping entries with indices.

#first
Returns the first element in the collection. An integer can be passed as an argument to return the first n elements.

#grep
One of the most important enumerables in Ruby. Returns an array of everything in the collection that matches the pattern supplied by #grep. A figurative "search" method.

#group_by
Returns a hash in which values are the elements of the original collection, grouped by parameters set in the block.

#inject
Combines elements of a collection by passing them through a black. Amazing for combining an array as a sum or product. A wonder tonic of an iterator.

#map
One step up from #each, returning a new array with the elements resulting from the block of code supplied. Super useful, always available.

#max
Returns the maximum value of a set.

#min
Returns the minimum value of a set. (Don't forget about these guys)

#select
Passes a block and returns all elements for which that block returns true.

#sort
Returns array with the elements sorted from lowest value to highest. A block can be supplied to alter what the array is sorted by.

These are just a handful of the most useful enumerables, but keep your eyes peeled for more. If you find yourself wondering if there's any easier way to do something in Ruby, there's probably an enumerable just for that!
